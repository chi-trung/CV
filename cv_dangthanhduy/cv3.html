Giới thiệu bản thân

Fullstack Developer với đam mê phát triển các ứng dụng web hiệu quả và bảo mật, đồng thời có kinh nghiệm trong việc phát triển ứng dụng di động Android. Có kinh nghiệm trong việc xây dựng và tối ưu hóa các hệ thống backend, đặc biệt là với Java và Spring Boot. Luôn tìm kiếm cơ hội để học hỏi và áp dụng các công nghệ mới.

Kỹ năng chuyên môn

Ngôn ngữ lập trình

C++

Đã nắm vững cú pháp cơ bản, các khái niệm lập trình hướng đối tượng (OOP) như kế thừa, đa hình, trừu tượng, và biết cách áp dụng hiệu quả để giải quyết các bài toán phức tạp. Đồng thời, có kiến thức sâu về cấu trúc dữ liệu và giải thuật, tối ưu hóa hiệu suất chương trình.

Java

Đã nắm vững cú pháp Java, hiểu rõ về JVM, lập trình hướng đối tượng và phát triển ứng dụng trên nền tảng Java. Đặc biệt có kinh nghiệm với các thư viện và framework phổ biến như Spring Boot để xây dựng các ứng dụng web mạnh mẽ và có khả năng mở rộng.

JavaScript

Đã nắm vững cú pháp JavaScript ES6+, thành thạo thao tác DOM để tạo ra các giao diện người dùng động và tương tác. Có kinh nghiệm với các thư viện frontend như React để xây dựng các component tái sử dụng và quản lý state hiệu quả.

Kotlin

Đã tìm hiểu sâu về cú pháp hiện đại của Kotlin, các tính năng lập trình hàm và khả năng tương tác với Java. Có kinh nghiệm trong việc phát triển ứng dụng Android Native, xây dựng giao diện người dùng với Android XML và Jetpack Compose.

Framework & Thư viện

Spring Boot

Đã tìm hiểu sâu về kiến trúc Microservices, phát triển các RESTful API mạnh mẽ, sử dụng Spring Data JPA để tương tác với cơ sở dữ liệu, và quản lý các dependency với Maven/Gradle. Có khả năng xây dựng các ứng dụng backend hiệu suất cao.

Django

Đã tìm hiểu về khung phát triển web nhanh của Python, bao gồm tích hợp ORM mạnh mẽ, hệ thống admin tự động, và cách xây dựng các ứng dụng web với kiến trúc MVT (Model-View-Template) một cách hiệu quả.

React

Đã tìm hiểu về xây dựng giao diện người dùng tương tác và hiệu quả với React.js, sử dụng JSX, quản lý state với Hooks (useState, useEffect) và Context API. Có khả năng phát triển các ứng dụng SPA (Single Page Application) hiện đại.

Node.js

Đã tìm hiểu về phát triển backend với JavaScript bằng Node.js, sử dụng các module và quản lý package với npm. Có khả năng xây dựng các server-side application, xử lý bất đồng bộ với Promises và async/await.

Express.js

Đã tìm hiểu về xây dựng API RESTful nhanh chóng và mạnh mẽ với Express.js, quản lý routing, middleware, và tích hợp với cơ sở dữ liệu. Có khả năng phát triển các ứng dụng web dựa trên kiến trúc client-server.

Phát triển di động

Lập trình ứng dụng di động Android (Native/Hybrid)

Am hiểu sâu về quy trình phát triển ứng dụng di động trên nền tảng Android, từ thiết kế UI/UX theo Material Design đến tích hợp các API (như bản đồ, thông báo), xử lý dữ liệu offline và tối ưu hóa hiệu suất ứng dụng.

Cơ sở dữ liệu

MySQL

Đã tìm hiểu chuyên sâu về thiết kế schema cơ sở dữ liệu quan hệ, tối ưu hóa truy vấn SQL, quản lý chỉ mục và sao lưu/phục hồi dữ liệu để đảm bảo hiệu suất và tính toàn vẹn.

PostgreSQL

Đã tìm hiểu về các tính năng nâng cao của PostgreSQL như JSONB, FTS (Full-Text Search), các loại dữ liệu tùy chỉnh, và tối ưu hóa hiệu suất truy vấn phức tạp cho các ứng dụng yêu cầu cao.

MongoDB

Đã tìm hiểu về cơ sở dữ liệu NoSQL dựa trên tài liệu, mô hình hóa dữ liệu linh hoạt, cách thực hiện các thao tác CRUD, và sử dụng Aggregation Framework để xử lý dữ liệu phức tạp trong các ứng dụng web hiện đại.

Firebase

Đã tìm hiểu về tích hợp Firebase Realtime Database và Cloud Firestore để đồng bộ hóa dữ liệu theo thời gian thực, quản lý xác thực người dùng (Authentication), và sử dụng Firebase Storage để lưu trữ tệp tin.

SQLite

Đã tìm hiểu về việc sử dụng SQLite trong các ứng dụng di động và nhúng, cách quản lý cơ sở dữ liệu cục bộ, thực hiện các truy vấn cơ bản và nâng cao để lưu trữ dữ liệu người dùng offline.

Công cụ & DevOps

Git

Đã tìm hiểu về quản lý phiên bản phân tán, các lệnh cơ bản (commit, push, pull, branch, merge), giải quyết xung đột, và quy trình làm việc nhóm hiệu quả trên GitHub/GitLab.

Docker

Đã tìm hiểu về container hóa ứng dụng, viết Dockerfile, xây dựng và quản lý Docker images, và triển khai ứng dụng trong môi trường Docker để đảm bảo tính nhất quán và khả năng mở rộng.

AWS

Đã tìm hiểu về các dịch vụ điện toán đám mây cơ bản của Amazon Web Services (AWS) như EC2 (máy chủ ảo), S3 (lưu trữ đối tượng), RDS (cơ sở dữ liệu quản lý), và VPC (mạng ảo riêng tư) để triển khai và quản lý ứng dụng.

Khác

RESTful API

Đã tìm hiểu sâu về nguyên tắc thiết kế RESTful API, các phương thức HTTP (GET, POST, PUT, DELETE), quản lý trạng thái, và cách xây dựng các API hiệu quả, dễ sử dụng và bảo mật cho các ứng dụng phân tán.

Microservices

Đã tìm hiểu về kiến trúc Microservices, cách chia nhỏ ứng dụng thành các dịch vụ độc lập, quản lý giao tiếp giữa các dịch vụ, và triển khai chúng một cách linh hoạt và có khả năng mở rộng.

CI/CD

Đã tìm hiểu về quy trình tích hợp liên tục (CI) và triển khai liên tục (CD), sử dụng các công cụ như Jenkins, GitHub Actions để tự động hóa kiểm thử, xây dựng và triển khai ứng dụng một cách hiệu quả và đáng tin cậy.
