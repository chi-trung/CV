Tính năng chính

• Hệ thống đăng ký và đăng nhập người dùng
• Tìm kiếm và lọc chuyến bay/khách sạn theo nhiều tiêu chí
• Quản lý đặt chỗ và thanh toán trực tuyến
• Hệ thống đánh giá và bình luận
• Quản lý lịch sử đặt chỗ và hóa đơn
• Thông báo qua email về trạng thái đặt chỗ

Công nghệ

• Backend: Java Spring Boot, RESTful API
• Frontend: HTML5, CSS3, JavaScript, Bootstrap
• Database: MySQL
• Security: Spring Security, JWT
• Payment: Integration với cổng thanh toán
• Email: JavaMail API

• Phát triển toàn bộ backend với Spring Boot
• Thiết kế và xây dựng cơ sở dữ liệu
• Tích hợp các API thanh toán
• Xây dựng giao diện người dùng
• Tối ưu hóa hiệu suất và bảo mật
